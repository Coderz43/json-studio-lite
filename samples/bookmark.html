Bookmark JSON

A practical JSON blueprint for saving bookmarks with tags, notes, and timestamps ‚Äî perfect for a bookmarking app or a personal knowledge base.

This JSON contains a meta section (owner, created/updated, default sorting) and a bookmarks array. Each bookmark has a unique id, required title and url, optional description and tags, and ISO-8601 timestamps. Add fields like favorite, folder, or favicon as needed.

Description:

This sample models a minimal, scalable bookmark database that‚Äôs easy to sort, filter and sync. You can extend it freely without breaking existing data.

  • meta ‚Äî Information about the list (owner, created/updated, default sorting).
  • bookmarks ‚Äî Array of bookmark objects.

Bookmark Object

  • id (number) ‚Äî unique identifier.
  • title (string, required) ‚Äî human-readable title.
  • url (string, required) ‚Äî absolute URL.
  • description (string) ‚Äî short note.
  • tags (array<string>) ‚Äî labels for grouping/filtering.
  • created_at / updated_at (string) ‚Äî ISO-8601 timestamps.
  • favorite (boolean) ‚Äî quick-access toggle.

Why this structure?

  • Portable between local apps, APIs, or static JSON.
  • Easy sorting by updated_at and tag-based filtering.
  • Future-proof: add folder, favicon, archived, etc.

Mini Schema (JSON Schema-style):

{
  "type": "object",
  "required": ["bookmarks"],
  "properties": {
    "meta": {
      "type": "object",
      "properties": {
        "name": {"type": "string"},
        "owner": {"type": "string"},
        "created_at": {"type": "string", "format": "date-time"},
        "updated_at": {"type": "string", "format": "date-time"},
        "sort": {
          "type": "object",
          "properties": {
            "by": {"type": "string", "enum": ["title","created_at","updated_at"]},
            "order": {"type": "string", "enum": ["asc","desc"]}
          }
        }
      }
    },
    "bookmarks": {
      "type": "array",
      "items": {
        "type": "object",
        "required": ["id","title","url"],
        "properties": {
          "id": {"type": "integer"},
          "title": {"type": "string"},
          "url": {"type": "string", "format": "uri"},
          "description": {"type": "string"},
          "tags": {"type": "array", "items": {"type": "string"}},
          "created_at": {"type": "string", "format": "date-time"},
          "updated_at": {"type": "string", "format": "date-time"},
          "favorite": {"type": "boolean"}
        }
      }
    }
  }
}
